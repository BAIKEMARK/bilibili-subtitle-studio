# 🚀 快速开始

## 第一次使用？请按以下步骤操作：

### 步骤 1️⃣：获取登录Cookie ⭐

**最简单的方法（推荐）：**
```bash
python setup_cookie.py
```

然后按提示：
1. 登录B站网站
2. F12 → Application → Cookies
3. 复制 SESSDATA 和 bili_jct 的值
4. 粘贴到脚本中

**其他方法：**
- 查看 [最简单的Cookie获取方法.md](最简单的Cookie获取方法.md)
- 或运行：`python get_cookie.py`（需要安装selenium）

### 步骤 2️⃣：提取字幕

```bash
python 字幕.py
```

选择模式：
- 输入 `1` - 提取单个视频字幕
- 输入 `2` - 批量提取多个视频字幕
- 输入 `3` - 手动粘贴JSON

### 步骤 3️⃣：查看字幕

字幕会自动保存为 `{视频ID}_字幕.txt` 文件
